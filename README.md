# Solar Smart Building

지도에서 건물 옥상을 선택하여 태양광 발전 가능성을 분석하는 웹 애플리케이션입니다.  
건물 면적을 기반으로 설비 용량, 연간 발전량, 전기요금 절감, 탄소 저감 효과를 계산합니다.

배포 링크: https://new-solar-building.vercel.app/

---

##  프로젝트 개요

이 프로젝트는 도시 내 건물 옥상에 태양광 설비를 설치했을 때의 효율을 분석하기 위해 제작되었습니다.

사용자는 지도에서 직접 폴리곤을 그려 분석 영역을 지정할 수 있으며,  
이를 기반으로 다음과 같은 결과를 제공합니다:

- 설치 가능 면적
- 예상 설비 용량 (kW)
- 연간 발전량 (kWh)
- 전기요금 절감 효과
- 탄소 배출 저감량
- 3개년 평균 발전량
- PR(성능비) 시나리오 비교

---

##  주요 기능

### 1. 건물 선택 방식
- 직접 그리기 (폴리곤 생성)

### 2. 태양광 발전량 분석
- 실제 3개년 기상 데이터를 활용하였으나, 발전량 계산은 단순화된 모델을 기반으로 수행하였다.

### 3. 결과 제공
- 연간 발전량
- 월 평균 발전량
- 특정수율 (kWh/kW·year)
- 최소 / 최대 발전량 (오차 포함)

### 4. 경제성 및 환경성 분석
- 전기요금 절감액
- 탄소 배출 감소량

#  계산 방식

### 1. 설치 가능 면적
건물 옥상의 전체 면적 중 약 55%만 태양광 설치가 가능하다고 가정

### 2. 설비 용량
설치 면적을 기준으로 단위 면적당 설비 용량을 적용하여 계산

### 3. 발전량 계산
설비 용량과 특정수율을 기반으로 연간 발전량을 산정

### 4. 특정수율
약 1200~1600 kWh/kW·year 범위의 값을 가정하여 사용

### 5. PR (성능비)
태양광 시스템의 실제 운영 손실을 반영하기 위해 PR(성능비)을 적용
- 보수적: 0.75
- 일반적: 0.80
- 이상적: 0.85

##  가정 및 한계

본 프로젝트는 실제 설계가 아닌 추정 모델이며 다음과 같은 가정을 포함합니다.

- 옥상의 55%만 설치 가능
- 건물 방향 및 경사 미반영
- 주변 건물에 의한 그림자 미반영
- 유지보수 손실 단순화
- 기상 조건은 단순화된 평균값 및 가정 데이터를 사용

따라서 실제 설치 결과와 차이가 있을 수 있습니다.
